# Breast Cancer Diagnosis Prediction using Machine Learning

## Project Overview
This project builds and compares multiple machine learning models to predict whether a tumor is malignant or benign using the Breast Cancer Wisconsin dataset.

## Models Implemented
- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest
- K-Nearest Neighbors (KNN)
- Naive Bayes
- Decision Tree

## Final Model
Tuned Logistic Regression (C=0.1)

## Results
- Accuracy: 98.25%
- False Negatives: 2
- ROC-AUC: ~0.99
- Stratified Cross Validation applied

## Techniques Used
- Data Preprocessing
- Correlation Analysis
- Model Comparison
- Hyperparameter Tuning (GridSearchCV)
- Stratified K-Fold Cross Validation
- Feature Importance Analysis

## Technologies
Python, NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn, Jupyter Notebook
